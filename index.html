E-commerce Microservice

This project is a Dockerized microservice application. It utilizes Spring Boot and Data JPA for creating the services using MongoDB and MySQL, Apache Kafka for asynchronous communication, WebClient to perform HTTP requests with circuit breaker configured, Netflix Eureka for service discovery and registry, a Gateway for centralized access, Keycloak for authentication and authorization, and Sleuth with Zipkin for distributed tracing.

Details Github

Reddit Clone

Full stack project with Spring and Angular using Spring Security (JWT), Spring Data JPA with MySQL as the database. The frontend is built with Angular using Bootstrap. Each new account created requires confirmation, which is sent via email. The logged-in user can access the endpoints for creating subreddits, posts within subreddits, and comments on posts. The user's profile includes a history of posts and comments made by the logged-in user.

Details Github

Bookstore Fullstack

Full Stack Project with Spring and Angular using Spring Security (JWT) with roles and Spring Data JPA with PostgreSQL as the database. The front-end is developed with Angular using Bootstrap. It simulates book purchases, where the purchase history can be found in the logged-in user's profile. Some endpoints can only be accessed by administrators, such as inserting new books into the catalog.

Microservice Study

Project for microservices study simulating the payment confirmation of an order using Spring Cloud. Utilization of Gateway, Service Discovery with Spring Cloud Netflix, synchronous communication with OpenFeign, endpoint testing with Postman, and MySQL for database.
